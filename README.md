# Prismaflow: Modular Prompt Studio

> **Structure. Randomize. Flow.**

**Prismaflow** (formerly Varia) is a professional prompt engineering environment for Generative AI. It treats prompts as modular structures composed of fixed constants and variable options blocks, enabling **Controlled Variance** in your creative workflows.

---

## ✨ Features

*   **Options Blocks:** Create `[variable | options]` blocks that randomize on every generation.
*   **Smart Normalization:** The editor automatically ensures valid cursor positions around blocks, preventing "trapped cursor" issues common in rich text editors.
*   **Library:** Persistent local storage for your prompt projects.
*   **Prismatic Labels:** Use color-coded labels to visually segment long prompts (e.g., `#SUBJECT`, `#LIGHTING`).
*   **Cyber-Minimalist UI:** A dark, distraction-free interface built for flow state.

## 🔮 Philosophy: Controlled Variance

**Prismaflow** is built for when you have a specific vision (Structure) but want to explore details (Nuance).

**Example:**
*   **Constant:** "Cinematic wide shot of a cyberpunk street..."
*   **Variable:** "...neon [blue | pink | green] lighting, raining [heavy | light | mist], featuring a [robot | cyborg | human]..."

Lock the structure. Flow with the variables.

# Prismaflow: 模組化提示詞工作室 (繁體中文)

> **結構化設計。隨機變數。流暢創作。**

**Prismaflow** 是一個專為生成式 AI 創作者打造的專業提示詞（Prompt）編輯環境。它將提示詞視為由「固定常數」與「變動選項」組成的模組，讓您在保持構圖穩定的同時，精準控制細節的隨機性。

## ✨ 核心功能

*   **選項模組 (Options Blocks):** 建立可隨機變化的關鍵字區塊，一鍵探索無限可能。
*   **智慧正規化 (Smart Normalization):** 編輯器會自動修正結構，確保游標永遠能在區塊間自由移動，解決了游標卡住的問題。
*   **專案圖書館 (Library):** 在側邊欄管理與切換您的所有專案。
*   **稜鏡標籤 (Prismatic Labels):** 使用色碼標籤將長串提示詞分段，提升可讀性。
*   **賽博極簡介面:** 專注於創作的深色系設計。
